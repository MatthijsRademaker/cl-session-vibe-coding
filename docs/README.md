## Reading Order

### First Time Teaching This Workshop?
1. **WORKSHOP_HANDS_ON.md** - Understand the flow
2. **INSTRUCTOR_SCRIPT.md** - Practice the delivery
3. **METRICS.md** - Know the data
4. **WHEN_NOT_TO_USE.md** - Understand limitations

### Preparing for a Session?
1. **PARTICIPANT_GUIDE.md** - Send to participants beforehand
2. **WORKSHOP_HANDS_ON.md** - Review timing and checkpoints
3. **Presentation slides** - See `/presentation/slides.md`

### Customizing the Workshop?
1. **WORKSHOP_GUIDE_2HR.md** - See customization tips
2. **CLAUDE.md** - Understand the architecture
3. **BRANCHING_STRATEGY.md** - Know the Git structure
